# Heart Disease Prediction App

A machine learning web app that predicts the risk of heart disease based on user health data. Built with Streamlit for an easy and interactive interface.

## Features

- Simple Streamlit UI
- Instant prediction
- User-friendly design
- Machine learning based results

## Dataset

Dataset obtained from Kaggle.

## Input Parameters

The model may use features such as:

- Age
- Sex
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol
- Fasting Blood Sugar
- Maximum Heart Rate
- Exercise Induced Angina
- Oldpeak
- Thalassemia

## Output

The application predicts:

- Presence of Heart Disease
- No Heart Disease
